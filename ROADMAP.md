# Laravel AI CRM - Features & Roadmap

## Current Features

Based on the existing implementation, this CRM supports the following core AI-powered features:

1. **Lead Scoring**
   - An AI-powered system that analyzes lead data to assign a priority score based on potential value. Factors such as source, behavior, and demographics are used for the calculation.

2. **Automated Email Drafting**
   - Automatically generates personalized email drafts using AI, referencing lead data and previous customer interactions. These drafts are highly customizable prior to sending.

3. **Sentiment Analysis**
   - Gauges customer satisfaction by processing and analyzing feedback through an AI sentiment model.

4. **Churn Prediction**
   - Identifies patterns in customer behavior to predict potential churn, providing proactive insights for better retention.
